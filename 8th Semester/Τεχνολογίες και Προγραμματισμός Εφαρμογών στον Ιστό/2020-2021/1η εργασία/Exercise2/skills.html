Σημαντικά Projects

Μερικά από τα project που εκπόνησα ατομικά ή ομαδικά ως μέρος των σπουδών μου είναι τα ακόλουθα:

  • Java/Android
    • "Publisher/Subsciber Σύστημα αναζήτησης και αναπαραγωγής τραγουδιών"

      Το συγκεκριμένο project αφορούσε ένα ολοκληρωμένο Pub/Sub σύστημα αναζήτησης τραγουδιών (παρόμοιο με το Spotify) και υλοποιήθηκε σε δύο στάδια.
      Σε πρώτη φάση υλοποιήθηκε σε Java η κύρια λειτουργικότητα της εφαρμογής με χρήση sockets.
      Προσομοιώθηκε η λειτουργία πολλών publishers και clients, αλλά και των ενδιάμεσων brokers που επεξεργάζονταν τα αιτήματα που δέχονταν παράλληλα με χρήση νημάτων.
      Για την κατανομή και την αναζήτηση των κλειδιών που αντιστοιχούν στους καλλιτέχνες μεταξύ των publishers αλλά και των brokers χρησιμοποιήθηκε consistent hashing.
      Στην επόμενη φάση ξεκίνησε η υλοποίηση της Android εφαρμογής που θα χρησιμοποιούσαν οι πελάτες.
      Αυτή περιείχε ολόκληρη τη λειτουργικότητα της μεριάς των clients καθώς και μια ολοκληρωμένη διεπαφή χρήστη.
      Έτσι η λειτουργία των publishers και των ενδιάμεσων κόμβων του δικτύου (brokers) συνέχιζε να πραγματοποιείται μέσω της γραμμής εντολών, όμως οι πελάτες πλέον επικοινωνούσαν με αυτούς μέσω της εφαρμογής που διέθεταν.

    • "Εφαρμογή χειρισμού φούρνου μικροκυμάτων"

      Το project αφορά την βελτίωση μιας υπάρχουσας (φυσικής) διεπαφής ενός φούρνου μικροκυμάτων.
      Η υλοποίηση του project χωρίστηκε σε 3 στάδια, χρησιμοποιώντας το Ελικοειδές Μοντέλο Ανάπτυξης.
      Στο 1ο στάδιο αναπτύχθηκε ένα πρωτότυπο της εφαρμογής μετά από συνεντεύξεις που πήραμε από τους χρήστες της συσκευής και από ερωτηματολόγια που τέθηκαν στους εν δυνάμη χρήστες.
      Στο 2ο στάδιο δημιουργήθηκε ένα βελτιωμένο ημι-λειτουργικό πρωτότυπο μετά από feedback που λάβαμε πάνω στο προηγούμενο.
      Τέλος αναπτύχθηκε μια πλήρως λειτουργική Android εφαρμογή που υλοποιούσε και επέκτεινε την λειτουργικότητα του πρωτοτύπου του προηγούμενου σταδίου και ήταν σύμφωνη με τις αρχές σχεδίασης μιας εύχρηστης διεπαφής.

  • C++
    • "Command line εργαλείο ορισμού φίλτρων σε φωτογραφίες"

      Το πρόγραμμα που αναπτύχθηκε σε αυτό το project αποτελούταν βιβλιοθήκες για την ανάγνωση, αποθήκευση και επεξεργασία οποιασδήποτε PPM εικόνας.
      Ο χρήστης προσδιορίζει μέσω της γραμμής εντολής τα φίλτρα που επιθυμεί να εφαρμοστούν και το πρόγραμμα είναι υπεύθυνο για την επεξεργασία των pixel που αποτελούσαν την εικόνα με τρόπο που όριζε κάθε φίλτρο. Τα φίλτρα που μπορούσαν να εφαρμοστούν σε μια εικόνα ήταν:

      1. Linear Filter (Επεξεργασία των χρωμάτων της εικόνας)
      2. Gamma Filter (Επεξεργασία της φωτεινότητας της εικόνας)
      3. Blur Filter (Θόλωση της εικόνας)

  • C
    • "Πρόγραμμα command line για κλείσιμο εισητηρίων με χρήση νημάτων"

      Το συγκεκριμένο πρόγραμμα προσωμείωνε τη λειτουργία ενός συστήματος διαχείρισης κρατήσεων και έκδοσης εισιτηρίων.
      Για κάθε πελάτη δημιουργείτο ένα νέο νήμα το οποίο δέσμευε θέσεις σε έναν πίνακα.
      Προκειμένου κάθε συναλλαγή να είναι ατομική χρησιμοποιήθηκαν κλειδαριές mutex ώστε η τροποποίηση του πίνακα να περιοριστεί σε ένα νήμα τη φορά
